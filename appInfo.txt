# BDMT01 어플리케이션 : 배드민턴 라켓 상세정보 및 후기 제공
  ※ 모든 대화는 한국어로 할 것
  ※ 프로젝트명 BDMT01
  ※ 프로젝트 구성시 확장성, 범용성 고려
  ※ 라켓 관련 모든 스펙 정보 제공


# 주의사항
  ※ .env 파일 보안 주의
  ※ 데이터 입출력 문제 시 supabase 테이블 / 권한 설정 / 함수 / URL Configuration 확인


# ADMIN 페이지
  ◯ 라켓 CRUD
  ◯ 태그 CRUD
  ◯ 건의사항 CRUD

# PROFILE 페이지
  ◯ 로그인한 사용자의 정보 표시
  ◯ 건의사항 입력
  ◯ 건의사항 목록 표시

# USER 페이지
  ◯ 라켓정보를 카드보기 형식으로 렌더링
  ◯ 사각형 카드 안에 라켓관련 다음의 정보를 제공(supabase 데이터 수신)
    - 카드보기시 가능하면 2열로 정렬하고 반응형일 것
    - 신분증 처럼 이미지와 정보 배치
      • 카드내 표기되는 영어는 무조건 대문자
    - 라켓이미지
    - 평점
      • 1인1평가
      • 평점과 참여자수만 표시 (예: ★ 4.7(26))
    - 태그
      • 일단은 관리자가 입력
  ◯ 라켓을 필터(브랜드, 무게, 밸런스, 탄성) 또는 라켓명 / #태그로 검색 가능
    - 초기화면은 데이터 전송량 최소화를 위해 빈화면으로 하며 필터 선택시 렌더링
  ◯ 별점 입력
    - 평균별점(참여자수) 표시
      • db함수 : get_racket_stats
    - 페이지 넘기지 않고 별점 바로 입력 가능

# 에디터 : 안티그래비티
# 프론트엔드 : vue3.js(composition API), 바닐라js
# 백엔드 : supabase
  ◯ Auth : 구글 로그인 구현 후 다른 방식 추후 구현
  ◯ Storage : BDMT01 - racket_storage 폴더
    - AI 라켓이미지 저장, 파일명 : 소문자 + 언더바 + 공백없음
  ◯ 테이블 -> sb_tables.txt
  ◯ DB함수 -> sb_db_functions.txt
  ◯ 접근정책 -> sb_auth_policies.txt
  ◯ 라우터 적절히 사용 : admin, user, login, profile...
