

◯ 배드민턴 라켓 상세정보 및 후기 제공
  - 라켓 관련 모든 스펙 정보 제공
  - 라켓 별 사용자가 키언어로 태그 추가 가능. 예를들어 #초보자용 #소리좋음 #잘깨짐 #가성비
  - 좋아요, 보통, 별로에요 기능
  - 라켓을 필터(브랜드별, 무게별 등), 또는 태그로 검색 가능
  - 라켓정보를 가져와 라켓정보페이지에 여러가지 보기 형식으로 보여줌. 카드형식, 자세히보기 등 컴포넌트를 만들어 렌더링
◯ 에디터 : 파이어베이스 스튜디오
◯ 프론트엔드 : vue3.js
◯ 백엔드 : supabase
  - 라켓별 브랜드, 길이, 무게, 밸런스, 탄성, 최대장력, 그립두께를 supabase 테이블에 저장
  - supabse 라켓테이블명 : rackets, 태그테이블명 : racket_tags, 라켓-태그 관계테이블명 : rackets-tags
◯ 스타일 : tailwind
◯ Auth : 일단 구글 로그인 구현
◯ github 사용
◯ Cloudflare 배포

◯ 웹 완성 후 하이브리드앱 출시를 고려하여 디자인


◯ 빠른 기능구현으로 어플 수요 파악 후 계속 진행 시 디테일 보완
  - 웹, 앱 빠른 앱 출시 후 앱 네이티브 전환 고려
  - 최소 필수 기능만 구현하여 출시 후 보완 고려
