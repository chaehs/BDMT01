# Supabase - Authentication - Policies

  ◯ profiles
    - 사용자는 본인의 프로필을 삭제할 수 있음 / DELETE / authenticated
    - 인증된 모든 사용자의 프로필 조회 허용 / SELECT / authenticated
    - 사용자 본인의 프로필 생성 허용 / INSERT / authenticated
    - 사용자 본인의 프로필 수정 허용 / UPDATE / authenticated

  ◯ racket_reviews

  ◯ racket_tag_map
    - 누구나 태그 매핑 조회 가능 / SELECT / public
    - 매니저 이상 태그 매핑 수정 가능 / ALL / authenticated

  ◯ rackets
    - 누구나 라켓 정보 조회 가능 / SELECT /	public
    - 매니저 이상 라켓 등록 가능 / INSERT /	authenticated
    - 매니저 이상 라켓 수정 가능 / UPDATE /	authenticated
    - 최고 관리자만 라켓 삭제 가능 / DELETE /	authenticated

  ◯ ratings
    - 별점 수정 허용 / UPDATE / authenticated
    - 별점 작성 허용 / INSERT / authenticated
    - 별점 조회 허용 / SELECT / public

  ◯ suggestions
    - 관리자 전권 / ALL	/ authenticated
    - 작성 권한 / INSERT	/ authenticated
    - 조회 권한 / SELECT	/ authenticated

  ◯ tags
    - 누구나 태그 목록 조회 가능 / SELECT / public
    - 매니저 이상만 태그 생성 및 수정 가능 / ALL / authenticated
